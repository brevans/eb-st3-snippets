# eb-st3-snippets
EasyBuild snippets for Sublime Text 3

to "install":

```
ln -s /path/to/eb-st3-snippets/snippets ~/.config/sublime-text-3/Packages/User/
#or
cp snippets/* ~/.config/sublime-text-3/Packages/User/
```
